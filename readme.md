# RSS Puzzle



## Description:
This project was inspired by Rolling Scopes School task [RSS Puzzle](https://github.com/rolling-scopes-school/tasks/tree/master/stage2/tasks/puzzle). The game provides an ability to improve user's language skills by constructing sentences with given sequence of words. Game consists of levels, each level has 5-10 sentences to complete. It is possible to listen to the current sentence, check russian translation. Gradually completing the sentences, the picture will be filled and thereby user will be closer to victory.
